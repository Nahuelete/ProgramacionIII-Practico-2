# 🎥 Cine "AURORA"

**Una función inolvidable.**

Mucho más que una pantalla. Un nuevo concepto de cine que celebra la diversidad visual, desde los grandes éxitos de taquilla hasta las joyas del cine independiente, todo bajo una atmósfera cálida que invita a quedarse después de los créditos.

---

## 📋 Descripción del Proyecto

Aurora consiste en una plataforma web profesional para un cine de estética minimalista. El proyecto abarca el diseño de la experiencia de usuario, un catálogo visual de películas, control del perfil del propio consumidor y adelanto de estrenos.

---

## ✨ Características

- ✅ Página de inicio con peliculas en cartelera
- ✅ Página con detalles de cada una de las peliculas
- ✅ Menu de registro
- ✅ Menu de inicio de sesion
- ✅ Página de próximos estrenos

---

## 🛠️ Tecnologías Utilizadas

- **HTML5** - Estructura y semántica
- **CSS3** - Estilos y diseño
